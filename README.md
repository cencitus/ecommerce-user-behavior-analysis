# E-commerce User Behavior Analysis

Пет-проект по продуктовой аналитике на данных интернет-магазина косметики.

Я взяла event-log датасет с Kaggle и попробовала разобрать, как пользователи проходят путь от просмотра товара до покупки: где они отваливаются, как меняется активность в течение месяца и какие гипотезы можно предложить для улучшения продукта.

## Данные

Источник: Kaggle — `eCommerce Events History in Cosmetics Shop`.
https://www.kaggle.com/datasets/mkechinov/ecommerce-events-history-in-cosmetics-shop?resource=download&select=2019-Dec.csv

В датасете каждая строка — это событие пользователя:

- `view` — просмотр товара
- `cart` — добавление товара в корзину
- `remove_from_cart` — удаление товара из корзины
- `purchase` — покупка

В проекте использовались данные за декабрь 2019 года.

Файл с данными не загружен в репозиторий, потому что он слишком большой. Для запуска ноутбука датасет нужно скачать с Kaggle.

## Что я делала

В проекте я:

- посмотрела структуру данных;
- проверила пропуски, дубликаты и странные значения в цене;
- удалила полные дубликаты;
- построила динамику активности пользователей по дням;
- построила воронку `view → cart → purchase`;
- посчитала конверсии между этапами воронки;
- посмотрела покупки, выручку и среднюю цену покупки;
- сравнила бренды по выручке, количеству покупок и средней цене;
- посчитала простой D1 и D7 retention;
- сформулировала выводы и продуктовые гипотезы.

## Основные результаты

Воронка по уникальным пользователям:

- `view`: 358 212 пользователей
- `cart`: 83 458 пользователей
- `purchase`: 25 613 пользователей

Конверсии:

- `view → cart`: 23.30%
- `cart → purchase`: 30.69%
- `view → purchase`: 7.15%

Retention:

- D1 retention: 7.36%
- D7 retention: 2.60%

Также я заметила, что к концу декабря снижается количество событий, пользователей, покупок и выручка. При этом средняя цена покупки растёт, поэтому падение выручки больше похоже на снижение количества покупок, а не на падение среднего чека.

## Выводы

Самый большой отток пользователей происходит между просмотром товара и добавлением в корзину. Это может быть связано с карточкой товара, ценой, недостатком информации, отзывами или удобством интерфейса.

Также часть пользователей добавляет товары в корзину, но не завершает покупку. Для этого этапа можно проверять гипотезы про напоминания о корзине, скидки, условия доставки или упрощение оформления заказа.

По брендам видно, что высокая выручка может появляться по разным причинам. Например, `runail` выглядит как массовый бренд с большим количеством покупок, а некоторые бренды попадают в топ выручки за счёт высокой средней цены.

## Ограничения

- Анализ сделан только за декабрь 2019 года.
- Декабрь может быть сезонным месяцем из-за праздников.
- В данных нет информации о стране пользователей, источниках трафика и маркетинговых акциях.
- В `category_code` очень много пропусков, поэтому я не делала основные выводы по категориям.
- В `brand` тоже есть пропуски, поэтому выводы по брендам относятся только к товарам, где бренд был указан.
- Retention считается только внутри одного месяца.
